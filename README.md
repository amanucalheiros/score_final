# Score Ascendente

Landing page em Vite + React + TypeScript, pronta para GitHub, Vercel ou Netlify.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

## Netlify

- Build command: npm run build
- Publish directory: dist
